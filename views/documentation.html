Moving forward, The NUESAKWASU API is organized around REST. Our API has predictable resource-oriented URLs, accepts form request bodies, returns JSON-encoded responses, and uses standard HTTP response codes, authentication, and verbs.

Routes

The available routes for this API include:

  • Staffs
  • PDFs(courses)

/staff Route

The posible requests for this routes include:GET, POST, PUT, DELETE.

Schema

```Staff Schema:

firstname: { type: String, required: true },
